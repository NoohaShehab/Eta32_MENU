## 📖 الملفات الموجودة

### 1. 🚀 **TEAM_SETUP.md** - دليل إعداد الفريق

**الوصف**: خطوات كاملة لأي عضو فريق جديد يريد البدء بالمشروع

**يشمل**:

- كيفية استنساخ المشروع
- تثبيت الأدوات المطلوبة
- فتح المشروع في VS Code
- تجميع ورفع البرنامج
- استكشاف الأخطاء الشائعة

**متى تستخدمه**: عند إضافة عضو جديد للفريق

---

### 2. ⚡ **QUICK_REFERENCE.md** - الأوامر السريعة

**الوصف**: مرجع سريع للأوامر الأساسية

**يشمل**:

- أوامر Git
- أوامر PlatformIO
- الاختصارات في VS Code
- Workflow اليومي
- حلول سريعة للمشاكل

**متى تستخدمه**: كمرجع أثناء العمل اليومي

### 3. 🤝 **CONTRIBUTING.md** - دليل المساهمة

**الوصف**: قواعد وإجراءات المساهمة في المشروع

**يشمل**:

- خطوات عمل feature جديدة

- أنواع المساهمات المختلفة
- قوائم الفحص

**متى تستخدمه**: قبل إضافة أي ميزة أو إصلاح

**متى تستخدمه**: عند فهم الكود أو إضافة ميزات
