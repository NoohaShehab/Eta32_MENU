# 🎮 دليل الاستخدام السريع - ProjectMp v2.0

## البدء السريع

### القائمة الرئيسية 🏠

عند التشغيل ستظهر:

```
==== MAIN MENU ====
1:Timer  2:Game
```

اختر من الخيارات:

- **اضغط 1** → الانتقال لوضع المؤقت
- **اضغط 2** → الانتقال للعبة الديناصور

---

## 🕐 وضع المؤقت (Timer Mode)

### الخطوات:

```
1. انتظر الرسالة "Enter Seconds:"
2. أدخل أول رقم (0-9)
3. أدخل ثاني رقم (0-9)
   النتيجة: 00-99 ثانية
```

### مثال:

```
إذا أردت 45 ثانية:
اضغط 4 ثم 5
```

### أثناء التشغيل:

- **الشاشة العلوية (LCD):** تظهر الوقت المتبقي
- **الشاشة السفلية (7-Segment):** تعديل ديناميكي للوقت
- **Buzzer:** صامت خلال العداد

### عند انتهاء الوقت:

```
الشاشة:    TIME'S UP!
الصوت:     3 بيبات سريعة + بيب واحد طويل
العودة:    الضغط على أي زر → القائمة الرئيسية
```

### الإلغاء أثناء التشغيل:

- **اضغط A أو D** → إلغاء المؤقت والعودة للقائمة

---

## 🎮 لعبة الديناصور (Dinosaur Game)

### الهدف:

تجنب العقبات (الشجرة/البلوك) بقدر الإمكان وجمع أكثر نقاط!

### التحكم:

- **أي زر على لوحة المفاتيح** = القفز
- **A أو D** = الخروج من اللعبة

### الشاشة:

```
السطر العلوي:  Score:05      (عدد العقبات التي تجنبتها)
السطر السفلي:  [طائر] ▓▓▓▓  (اللاعب والعقبة)
```

### آلية اللعب:

1. **الكائن ينزلق من اليمين لليسار** (العقبة)
2. **اللاعب يبدأ على الأرض**
3. **عند الضغط على أي زر** → اللاعب يقفز للأعلى
4. **عند انتهاء الضغط** → اللاعب يسقط للأسفل
5. **كل عقبة تمر = +1 نقطة**

### الاصطدام:

```
إذا لم تقفز في الوقت المناسب:
الشاشة:    GAME OVER!
           Score: 05
الصوت:     3 بيبات
العودة:    الضغط على أي زر → القائمة الرئيسية
```

## 🔑 لوحة المفاتيح (Keypad Layout)

```
+-----+-----+-----+-----+
| 1   | 2   | 3   | A   | ← Select Timer/Game + Jump
+-----+-----+-----+-----+
| 4   | 5   | 6   | B   |
+-----+-----+-----+-----+
| 7   | 8   | 9   | C   |
+-----+-----+-----+-----+
| *   | 0   | #   | D   | ← Exit
+-----+-----+-----+-----+
```

**الأزرار المهمة:**

- **1**: اختر المؤقت
- **2**: اختر اللعبة
- **0-9**: إدخال الأرقام
- **A**: قفز / خروج
- **D**: قفز / خروج

## 📊 التدرجات والمستويات

### السرعة:

- **اللعبة**: عقبة جديدة كل 250ms
- **المؤقت**: تحديث كل 1000ms

### المستويات الممكنة (ميزة مستقبلية):

```
Level 1: 300ms (سهل)
Level 2: 250ms (متوسط)  ← الحالي
Level 3: 200ms (صعب)
Level 4: 150ms (صعب جداً)
```

## 🔊 الأصوات والمؤثرات

| الحدث           | الصوت                |
| --------------- | -------------------- |
| القفز في اللعبة | بيب قصير (50ms)      |
| انتهاء المؤقت   | 3 بيبات + 1 بيب طويل |
| نهاية اللعبة    | 3 بيبات              |
| لا يوجد أخطاء   | بيب واحد             |

## 🐛 استكشاف الأخطاء

### المشكلة: لا تظهر الشاشة

- تأكد من توصيل LCD بشكل صحيح
- تحقق من الدبابيس: 25, 26, 27, 28, 29, 30

### المشكلة: الأزرار لا تستجيب

- تحقق من توصيل Keypad
- تأكد من دبابيس الصفوف (20-23) والأعمدة (10-13)

### المشكلة: الـ 7-Segment لا يعمل في المؤقت

- تحقق من الدبابيس 22, 23 (تفعيل الأرقام)
- تحقق من توصيل Port A (الأرقام نفسها)

### المشكلة: Buzzer لا يعمل

- تحقق من الدبس 21 (PC5)
- تأكد من وجود GND منفصل للـ Buzzer

## 💡 نصائح وحيل

1. **في اللعبة**: اضغط في اللحظة الأخيرة قبل العقبة
2. **الدقة أهم من السرعة** في تجنب العقبات
3. **استرح عند المؤقت**: لا داعي للقفل طول الوقت
4. **احفظ أفضل نقطة**: حاول التفوق على النقاط السابقة

## ⚠️ ملاحظات هامة

1. **لا تستخدم المفاتيح بسرعة جداً** - قد لا تستجيب
2. **انتظر 30ms بعد الضغط** للـ debounce
3. **إذا توقفت اللعبة**: اضغط A أو D
4. **احذر من الضغط المتكرر** على نفس الزر
